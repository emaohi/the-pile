# thePile

A personal learning system that transforms passive content consumption into intentional, engaged learning.

## Quick Start

**REQUIRED:** Use `superpowers:executing-plans` skill to implement this project.

## Documentation

- **Design doc:** `docs/thepile-lean-v1-design-doc.md` - Full product spec, screens, data model, algorithms
- **Implementation plan:** `docs/2025-12-20-lean-v1-implementation.md` - Phase-by-phase build instructions

## Tech Stack

- Next.js 14 (App Router, Server Actions)
- TypeScript
- Prisma + Turso (SQLite)
- Claude API for AI features
- Tailwind CSS + shadcn/ui

## Development Conventions

- **Never commit directly to master** - use feature branches per phase
- Follow the implementation plan phases in order
- Verify each phase before moving to next
